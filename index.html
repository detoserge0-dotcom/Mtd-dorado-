💡 Cette fonctionnalité nécessite un identifiant Google (« Client ID OAuth ») propre à la MTD, à créer une seule fois gratuitement sur Google Cloud Console : créez un projet → activez l'« API Google Drive » → créez un identifiant OAuth de type « Application Web » → ajoutez l'adresse de ce site dans « Origines JavaScript autorisées » → collez le Client ID obtenu ci-dessous. Cette configuration ne se fait qu'une seule fois.
